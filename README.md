# AI News Aggregator

A simple, automated AI news aggregator that fetches and displays AI-related news from top sources. Updates daily and deploys automatically to GitHub Pages.

## Features

- 📰 Aggregates news from multiple AI sources (OpenAI, Anthropic, Google AI, DeepMind, Hugging Face, and more)
- 🎨 Clean, responsive design that works on desktop and mobile
- 🔄 Automatically updates daily via GitHub Actions
- 📱 Easy to share with colleagues via a public URL
- ⚙️ Configurable sources and categories

## Quick Start

### 1. Create a GitHub Repository

1. Go to [GitHub](https://github.com) and create a new repository
2. Name it `ai-news-aggregator` (or any name you prefer)
3. Make it public (required for GitHub Pages)
4. Don't initialize with README (we'll push our existing files)

### 2. Upload the Project

From your local `ai-news-aggregator` directory:

```bash
cd /Users/joseph.gardner/ai-news-aggregator

# Initialize git repository
git init

# Add all files
git add .

# Commit
git commit -m "Initial commit: AI News Aggregator"

# Add your GitHub repository as remote (replace with your username and repo name)
git remote add origin https://github.com/YOUR_USERNAME/ai-news-aggregator.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. The workflow will run automatically and deploy your site

### 4. Access Your News Aggregator

Your site will be available at:
```
https://YOUR_USERNAME.github.io/ai-news-aggregator/
```

The first deployment may take a few minutes. After that, it will update automatically every day at 9 AM UTC.

## Customization

### Adding or Removing News Sources

Edit `config.json` to customize which RSS feeds are included:

```json
{
  "feeds": [
    {
      "name": "Source Name",
      "url": "https://example.com/feed.xml",
      "category": "Category Name"
    }
  ],
  "max_articles_per_feed": 5,
  "days_to_show": 7
}
```

**Categories:**
- `Company News` - Official announcements from AI companies
- `Research & Tools` - Technical research and tool releases
- `Community` - Community-driven content and analysis

### Changing Update Schedule

Edit `.github/workflows/update-news.yml` and modify the cron schedule:

```yaml
schedule:
  - cron: '0 9 * * *'  # Daily at 9 AM UTC
```

Examples:
- `0 */6 * * *` - Every 6 hours
- `0 12 * * *` - Daily at noon UTC
- `0 9 * * 1-5` - Weekdays only at 9 AM UTC

### Customizing the Design

The HTML design is generated by `fetch_news.py`. To customize:

1. Edit the CSS styles in the `generate_html()` function
2. Modify colors, fonts, spacing, etc.
3. Push changes to trigger a rebuild

## Manual Updates

You can manually trigger an update anytime:

1. Go to your repository on GitHub
2. Click **Actions** → **Update AI News**
3. Click **Run workflow** → **Run workflow**

## Troubleshooting

### Workflow fails

- Check the Actions tab in GitHub for error messages
- Ensure GitHub Pages is enabled in Settings
- Verify all RSS feed URLs in `config.json` are valid

### No articles appear

- Some feeds may be temporarily unavailable
- Check the `days_to_show` setting in `config.json`
- Verify feed URLs are correct

### Page not updating

- Check the last workflow run in the Actions tab
- Ensure the cron schedule is correct
- Manually trigger a workflow run to test
